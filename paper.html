What is ICCV2017 Paper Search Tool? [github]

This tool can search ICCV2017 papers on CVF open access.
The differences from open access are,

  1. Each paper has embedded vector generated by facebook fasttext, so that you can explore similar papers.
  2. At CVF open access, search keyword only hits in paper titles. But it also hits in abstracts in our tool.
  3. For multiple keywords, our search results will contain all the keywords and be sorted by word frequency.

This page is hosted by Google App Engine by free and the source code is in github!
